## 如何使用
1. 下載[專案的壓縮檔](https://github.com/davis-0378/remote_DatabaseHW/archive/refs/heads/main.zip)
2. 解壓縮 zip 檔案
3. 打開可編輯php的編輯器
4. 找到 index.php 並執行於瀏覽器
5. 接下來就進入系統介面操作
